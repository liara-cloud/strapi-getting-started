# Getting started with Strapi
## Deploying
- [Create New Node App](https://console.liara.ir/apps/create) & install the [Liara CLI](https://docs.liara.ir/cli/install)
- Set ENVs on Liara
- Create the disks mentioned in liara.json file (media & database)
```
git clone https://github.com/liara-cloud/strapi-getting-started.git
```
```
cd strapi-getting-started
```
```
liara deploy --platform node --port 3000
```
